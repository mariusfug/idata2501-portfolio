# Demo Service
Example REST service built with spring boot

## Requirements

- Java 1.7
- Maven 3.X

## Install

1. git clone https://github.com/starmer/demo-service.git
2. mvn package && java -jar target/demo-service-0.1.0.jar
3. Open a browser or curl localhost:8080
